PC Setup Guide

Follow these two steps to connect the QuickCut mobile app to your PC.

1

Create a Windows Local Account

Windows Settings
Already have a local account? If you see a "Local account" listed under Other users (step ②), you can skip this section and go to Section 2.
1

Open Settings → Accounts → Family & other users

Click ① Family & other users in the left sidebar, then click ② Add someone else to this PC. A Microsoft account dialog will open — click ③ I don't have this person's sign-in information.

① Accounts sidebar  →  ② Add someone else  →  ③ No sign-in info
2

Add a user without a Microsoft account

On the next screen click ① Add a user without a Microsoft account.

① Add a user without a Microsoft account
3

Enter a username and password

Fill in ① the username and password, then click ② Next.
Remember this username — you will enter the same name in the QuickCut mobile app's SMB Settings.

① Enter username & password  →  ② Next
4

Account created

The new local account now appears in the list.

New local account visible in the list
2

Set Up SMB Share for the QuickCut Folder

File Explorer
1

Open the QuickCut folder properties

Right-click the QuickCut folder → Properties. Click ① the Sharing tab, then click ② Advanced Sharing.

① Sharing tab  →  ② Advanced Sharing
2

Enable sharing and open Permissions

Check ① Share this folder. Confirm the share name is QuickCut ②. Click ③ Permissions to open the permissions window.

In the Permissions window, if your account already appears in Group or user names, skip steps ④⑤⑥. Otherwise click ④ Add, type your Windows account name in the text box ⑤ (the same name you created in Section 1), and click ⑥ OK.

Not sure of your account name? Check Section 1 Step 4 — it's the name shown under "Local account".
① Share this folder  →  ② Share name: QuickCut  →  ③ Permissions  →  ④ Add  →  ⑤ Enter account name  →  ⑥ OK
3

Grant Full Control and apply

Select your account ① and make sure Full Control – Allow ② is checked. Click ③ Apply → ④ OK to close the Permissions window, then click ⑤ Apply → ⑥ OK in the Advanced Sharing window, and finally ⑦ OK in the Properties window.

① Select account  →  ② Full Control Allow  →  ③ Apply  →  ④ OK  →  ⑤⑥⑦ Close all windows
✅

All done!

Open the QuickCut mobile app → SMB Settings, enter your PC's IP address and the Windows account name you just created. The connection status will change to Active.
